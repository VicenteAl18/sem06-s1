Desventajas de CCS plano

CSS plano genera códigos excesivos y muchas veces difíciles de comprender.

¿Qué es un pre - procesador?

Es un programa que te permite generar CSS a partir de la syntax única del preprocesador, la mayoría de preprocesadores CSS añadiran algunas características que no existen en CSS puro como anidamiento, nesting, modularidad, entre otros. Estas características hacen la estructura de CSS más legible y fácil de mantener.

¿Qué es SASS?

Sass es un preprocesador CSS es una herramienta que nos permite permite tener una organización modular de los estilos, lo cual es vital para proyectos grandes y reduce el tiempo para crear y mantener el CSS.
